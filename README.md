# Cipher Memorize Game

A simple webpage intended to help practice recognition and quick recall of classical ciphers and other elements commonly encountered in Puzzle Hunts. It’s a work in progress.

For entertainment purposes only: **in actual competitions you are, of course, free to use external resources**. This project is simply a personal training aid to improve speed of recognition and recall.

I’m careful to emphasize that I do not want anyone who views this repository to assume I’m making Puzzle Hunts to become exam-oriented / test-prep focused.
> i.e., reducing an activity to rote techniques and score-maximizing tactics rather than creative exploration.

I want this project to be a light, informal training tool, not a methodical or overly instrumental approach to puzzle competitions.

***

# 密码记忆游戏

这是一个用于帮助快速识别和记忆古典密码和其他Puzzle Hunt常见元素的简陋网页，仍在建设中。

实际比赛中完全可以使用外部资源（密码表、搜索引擎等）而非记忆来解谜，因此本项目只是类似于一种日常游戏，也可以使玩家在见到相应密码和谜题元素等时一定程度上更快地反应。

需要强调的是，本项目完全没有将Puzzle Hunt应试化的意图，也并非一个系统的训练工具，至于玩家如何看待与使用则完全随意。
